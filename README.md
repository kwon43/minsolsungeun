# 🔬 한국나노기술원 데이터를 활용한 반도체 공정 & 수율 탐구 웹앱

이 웹앱은 당곡고등학교 학생들이 실제 공공데이터를 기반으로 반도체 핵심 공정과 수율(Yield)의 관계를 쉽고 재미있게 탐구할 수 있도록 제작된 교육용 Streamlit 애플리케이션입니다.

---

## 👥 조원 및 역할 소개

우리 조는 반도체 공정 데이터를 수집 및 가공하고, 이를 사용자가 쉽게 이해할 수 있는 웹 기반 시각화 플랫폼으로 구현하는 역할을 나누어 협동 학습을 진행했습니다.

| 학번/이름 | 주요 역할 | 수행 내용 |
| :--- | :--- | :--- |
| **20302 권민솔** | **데이터 탐색 및 AI 사용** | - 공공데이터포털에서 한국나노기술원의 신뢰성 높은 반도체 공정 데이터 발굴<br>- 생성형 AI(대형 언어 모델)를 활용하여 코드 생성 |
| **20414 배성은** | **깃허브로 웹앱 구성** | - Git 및 GitHub 리포지토리 환경 구축 및 버전 관리<br>- Streamlit Community Cloud와 GitHub 저장소를 연동하여 웹앱 실시간 웹 호스팅 및 배포 담당 |

## 📅 데이터 출처 (Data Source)

본 애플리케이션은 국가에서 공식 개방한 공공데이터를 기반으로 신뢰성 있게 제작되었습니다.

* **데이터셋명**: 한국나노기술원_공정 정보
* **제공기관**: 과학기술정보통신부 한국나노기술원(KANC)
* **다운로드 출처**: [공공데이터포털 (data.go.kr)](https://www.data.go.kr/)
* **데이터 설명**: 한국나노기술원의 나노소자팹(Fab) 연구공정장비와 연계된 실제 나노소자 제작, 분석, 측정 세부 공정 목록입니다.

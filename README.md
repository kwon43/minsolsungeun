# 🔬 한국나노기술원 공정 데이터 기반 반도체 공정 & 수율 탐구 웹앱

이 웹앱은 당곡고등학교 학생들이 반도체의 주요 공정을 이해하고, 실제 연구소 데이터를 바탕으로 반도체 수율(Yield) 관리에 필요한 검사 및 분석 공정을 탐구할 수 있도록 제작된 교육용 Streamlit 애플리케이션입니다.

## 📁 프로젝트 구조
- `app.py`: 메인 페이지 (웹앱 소개 및 공정 개요)
- `pages/`: 멀티페이지 폴더
  - `1_Process_Explorer.py`: 전체 반도체 공정 그룹 탐색기
  - `2_Yield_And_Inspection.py`: 검사/계측 공정과 수율 간의 관계 분석 페이지 (중요 공정과 수율)
- `requirements.txt`: 필요한 라이브러리 목록
- `한국나노기술원_공정 정보_20260226.csv`: 한국나노기술원 실제 공정 데이터 파일

## 🚀 배포 방법
1. GitHub에 이 저장소를 업로드합니다.
2. [Streamlit Community Cloud](https://share.streamlit.io/)에 가입하고 저장소를 연결합니다.
3. 메인 파일 주소를 `app.py`로 지정하여 배포합니다.

# 🔬 한국나노기술원 데이터를 활용한 반도체 공정 & 수율 탐구 웹앱

이 웹앱은 당곡고등학교 학생들이 실제 반도체 공정 데이터를 기반으로 반도체 핵심 공정과 수율(Yield)의 관계를 쉽고 재미있게 탐구할 수 있도록 제작된 교육용 Streamlit 애플리케이션입니다.

---

## 📅 데이터 출처 (Data Source)

본 애플리케이션은 프로젝트 저장소 내에 직접 업로드하여 연동한 실제 연구소 데이터를 기반으로 구동됩니다. 아래 링크를 클릭하시면 저장소 내부의 실제 원본 데이터 화면으로 즉시 연결됩니다.

* **연동 데이터 파일 (클릭 시 원본 데이터로 이동)**: https://www.data.go.kr/data/15138469/fileData.do
* **데이터 설명**: 과학기술정보통신부 산하 한국나노기술원(KANC)에서 실제 나노소자팹(Fab) 연구공정장비로 진행하는 소자 제작, 표면 분석, 계측 및 신뢰성 측정의 세부 공정 마스터 정보가 담겨 있는 프로젝트 핵심 데이터베이스 파일입니다.
